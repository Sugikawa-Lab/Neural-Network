# NeuralNetwork(ニューラルネットワーク)

## 1. はじめに
- ニューラルネットワークとは **人間の脳の神経回路** を模倣した機械学習モデル。
- 複雑なデータの関係性を捉えることができる。


## 2. ニューラルネットワーク
　ニューラルネットワークは **微分可能な変換を繋げて作られた計算グラフ** である。円で表された **ノード(node)** に値が入り、ノードとノードが **エッジ(edge)** により繋がっている。<br>
　ノードの縦方向の集まりのことを **層(layer)** と呼び、 **ディープラーニング(deep learning)** とは、層の数が非常に多いニューラルネットワークを用いた機械学習の手法や、その周辺の研究領域のことを指す。

### ニューラルネットワークの歴史
|時期|事象|
|----|----|
|1943年|ウォーレン・マカロックとウォルター・ピッツが「神経回路網理論」(形式ニューロン)を考察。|
|1958年|心理学者のフランク・ローゼンブラッドが、最初に実装されたNNの一つである、 **パーセプトロン** について記述。|
|1967年|甘利俊一が多層パーセプトロンの確率的勾配降下法による学習を論理的に解析した論文を発表。|
|1986年|デビッド・ラメルハートらが出力側から偏微分する方法を **誤差逆伝播法(back propagation)** と命名。|
|2006年|ジェフリー・ヒントンらにより **オートエンコーダ** および **ディープビリーフネットワーク(deep belief network)(DBN)** が提案される。|
|2012年|GPU,および分散コンピューティングによる計算能力の向上で、大規模なネットワークを使用できるようになる。|
|2010年代| **ディープニューラルネットワーク(deep neural network)(DNN)** が音声、画像において高い性能を記録。|
|2014年|Googleにより **Seq2Seq** が開発される|
|2017年|「Attention Is All You Need」が発表され、seq2seqが抱える問題を解決する。これが現代の **Transformer** につながる。|

- 2010年前後まで研究は2つの要因から発展せず忘れられてしまう。
  1. 2層程度ではうまくいくが、多層化しても汎化性能が上がらず、 **過剰適合(overfitting)** が壁になった。
  2. ニューラルネットワークは構造的な自由度が高い一方、その選定が最終的な性能とどう結びつくのかはっきりとしなかった。
- しかし、画像を対象とする **畳み込みニューラルネットワーク(convolutional neural network)(CNN)** は、80年代後半には5つの層からなる多層ネットワークの学習に成功していた。
<br>
- Hintonらの **DBN**の研究がその後の流れを変える。
- DBNの学習を **事前学習(pretraining)** と呼ばれる方法で成功させる。
  - 具体的にはDBNを、層ごとに **制約ボルツマンマシン(restricted Boltzmann machine)(RBM)** と呼ばれる単層ネットワークに分解したうえで、これらRBMを入力層に近い側から順番に教師なしで学習する方法。
- その後、RBMの代わりに **自己符号化(autoencoder)** を用いることで、同様の層ごとの事前学習を行うことができるようになり、目標とする多層ネットワークの学習が可能となることが確認された。
- その後、 **DNN**が音声認識と画像認識のベンチマークテストで高い性能を示し、次々と過去の記録を塗り替えた。
- こうして、 **深層学習(ディープラーニング)** は広く認知されることになる。

- これらの発展から、多くの研究者が参加することで研究が進み、上述の事前学習とは異なる、DNNの学習の困難さを緩和する方法を生み出された
  - ReLU: 活性化関数
  - ドロップアウト: 正則化の一手法
  - バッチ正規化
  - 残渣接続
  - 勾配降下法の改良手法
    - RMSProp
    - Adam

- **注意機構(attention mechanism)** 
  - 機械翻訳の研究において発案される
  - 入力が要素の系列や集合であるとき、それら各要素の重要度を算出し、各要素を重み付ける計算を行う。

- **トランスフォーマー(Transformer)** 
  - 注意機構を構造の中心に据えたDNN
  - 自己教師学習との組合せで自然言語の分野を席巻
  - 最近では画像やその他の分野でも重要なDNNとなりつつある。

- **帰納バイアス(inductive bias)**
  - 問題やデータが持つ固有の構造
  - 帰納バイアスに合わせてDNNの構造をデザインすることで、学習を成功させ、推論の性能向上を狙う考え方がある。

### 課題
- 現状、可能になったのは **認知的自動化(cognitive automation)** である。
  - これは、あらかじめ定めた入力から出力へ一対一の写像を、DNNがデータの学習を通じ再現しているに過ぎない。
  - **自動化それ自体は「知能」からは遠い**

- 自動化と割り切った場合も、過剰適合が生じているかを見極めるのは困難。
  - 背景として、DNNが学習時、訓練データ内でのみ辻褄が合う手がかりをみつけてしまう、逆説的な「性能の高さ」がある。
  
- このような困難さを克服するため、様々な挑戦が行われている。

  - **自己教師学習(self-supervised learning)**
    - 入力に対し正解が容易に手に入る問題(プレテキストタスク)を学習することを指し、それによって入力データのよい表現を学習することを狙う方法。
    - 一般にはその後、目的とするタスクの学習を、比較的少量の訓練データを用いた転移学習によって行う。
    - 大きな成功として、トランスフォーマーを用いた **言語モデル(language model)** がある。

## 3. 層(layer)
　最初の層を **入力層(input layer)** 、最後の層を **出力層(output layer)** といい、その間にある層を **中間層(intermediate layer)**  もしくは **隠れ層(hidden layer)** という。<br><br>
　ニューラルネットワークは、層から層へと値を変換する。そのため、ニューラルネットワークはこの変換がいくつも連なってできる **一つの大きな関数** だと考えることができる。つまり、基本的に入力を受け取り、何か出力を返すものである。<br><br>
　入力層のノードの数は入力変数の数 **M** によって決定される。層と層の間にあるノード間の結合は、一つ一つが重みを持つ。 **全結合型ニューラルネットワークの場合、それらの重みをまとめて、一つの行列で表現する。** この際、バイアスをその重み行列に含めて扱うため、入力層の最後に常に1を持つノードが追加される。ノード数の計算式は以下の通りである。
```math
入力層が三つの場合 \\
x_1, x_2, x_3 (xは各入力層)\\
M = 3 \\
M + 1 = 3 + 1 = 4 \\
よって、ノード数は4 \\
```
　出力層のノード数を変化させることによって、いくつものカテゴリ数の分類問題にも対応することができる。また、離散的なカテゴリではなく、連続的な実数値を予測する回帰問題の場合、 **目標値の種類に合わせて出力層のノード数を決定** する。

## 4. 構造(architecture)
　層の数が同じでも、中間層のノード数を3にするか、5にするか、などには任意性がある。こういった値をどのように決めるかは、ニューラルネットワークを設計する人に委ねられる。また、中間層自体をいくつ重ねるか、その層数も自身で決める必要がある。このような自身で決定するパラメータを、 **ハイパーパラメータ** と呼ぶ。

## 5. ニューラルネットワークの計算
### 1. ユニットと活性化関数
　ニューラルネットワークを構成する最小の要素が **ユニット** である。<br>
  ユニットは、複数の入力を受け取り、一つの出力を計算する。例として、入力が $x_{1}$, $x_{2}$ ,  $x_{3}$の三つの場合、ユニットが受け取る総入力 $u$ は以下のようになる。

```math
u = w_{1}x_{1} + w_{2}x_{2} + w_{3}x_{3} + b
```

　$w_{1}$, $w_{2}$,$ w_{3}$ のように各入力に掛けられる数値を **重み(weight)** という。重みは入力に応じて異なる値を加える。また、それらをすべて加算した後、 **バイアス(bias)** と呼ばれる値 $b$ を加算する。<br>
　このようなユニットは、このユニットの総入力 $u$ を **活性化関数(activation function)** と呼ばれる関数 $f$ に入れて得られる値

```math
z = f(u)
```

を出力する。<br>
　複数のユニットが並べられ、それぞれ等しい数の入力があたえられた場合を考えてみる。具体的に、今回は二つのユニットが、それぞれ三つの入力を受け取ったとする。入力をインデックス $i = 1, 2, 3$ で表し、二つのユニットを $j = 1, 2$ で表すことにする。入力とユニット間の結合は $2 * 3 = 6$ 本あり、それぞれ異なる重み $w_{ji}$ が与えられるとする。つまり、二つのユニットが受け取る総入力は以下の通りである。

```math
u_{11} = w_{11}x_{1} + w_{12}x_{2} + w_{13}x_{3} + b_{1} \\
u_{12} = w_{21}x_{1} + w_{22}x_{2} + w_{23}x_{3} + b_{2} 
```
　この式は、入力ベクトル$x$、重み行列$W$、バイアスベクトル$b$、出力$u$を

```math
x = \begin{bmatrix}
        x_{1}\\
        x_{2}\\
        x_{3}\\
      \end{bmatrix}
```

```math
W = \begin{bmatrix}
        w_{11} & w_{12} & w_{13}\\
        w_{21} & w_{22} & w_{23}\\
        \end{bmatrix}
```

```math
b = \begin{bmatrix}
        b_{1}\\
        b_{2}\\
        \end{bmatrix}
```

```math
u = \begin{bmatrix}
        u_{11}\\
        u_{12}\\
        \end{bmatrix}

```

と定義すれば、

```math
u = Wx + b\\
z = f(u)
```

と書くことができる。<br>

### 2.活性化関数
　ユニットの総入力に使用される活性化関数は、様々な種類があり、目的に応じて使い分けられる。中でも最も基本となるのは、 **ReLU(rectified linear unit)** と呼ばれる次の関数である。

```math
f(u) = max(u, 0)
```

これは、恒等関数 $z=u$ のうち、 $u<0$ の部分を $u=0$ で置き換えてできる関数である。<br>
<br>
　なお、そのほかの関数として、以下のような活性化関数がある。
- シグモイド関数
  - ロジスティクスシグモイド関数：$f(u)=\frac{1}{1+e^{-u}}$
- 双極性正接関数：$f(u) = tanh(u)$
- マックスアウト
- ELU
- GELU

### 3.順伝播型ネットワーク
　ここまで取り扱ってきたネットワークに、層(layer)で説明した入力層、中間層、出力層を用いて、入力から出力へと一方向にのみ情報が向かい、フィードバックをもたない構造をもつニューラルネットワークを、 **順伝播型(ニューラル)ネットワーク(feed-forward neural network)** という。<br>
<br>
　計算を行う場合は、各層のユニットの入出力を区別するために、各変数の右肩に層の番号 $l(l=1, 2, 3, ...)$ をつけ、 $u^{(l)}$ や $z^{(l)}$ のように書くとする。なお、 $l$ の数は層の数によって増減する。<br>
　任意の層数 $L$ に一般化した場合、層 $l+1$ にユニットの出力 $z^{(l+1)}$ は、一つ下の層 $l$ のユニットの出力 $z^{(l)}$ から

```math
u^{(l+1)} = W^{(l+1)} z^{(l)} + b^{(l+1)} \\
z^{(l+1)} = f(u^{(l+1)}) 
```

のように計算される。以下では、ネットワークの最終的な出力を

```math
y ≡ z^{(L)}
```

と表記する。<br>
<br>
　順伝播型ネットワークが入力 $x$ から出力 $y$ を得る計算は、関数 $y = y(x)$ として表現できる。関数の中身は、各層間の結合の重み $W^{(l)}(l = 2, ... , L)$ とユニットのバイアス $b^{(l)}(l = 2, ..., L)$ によって決まる(パラメータ)。全パラメータを成分に持つベクトル $w$ を定義し、 $y(x;w)$ と書くことにする。

### 4.学習の概要
　順伝播型ネットワークは一つの関数 $y(x;w)$ を表現し、これはネットワークのパラメータ $w$ 次第で変化する。目標とする関数は、その具体的な姿かたちはわからないものの、その入出力のペアが複数与えられているとする。すなわち、入力 $x$ に対する望ましい出力 $d$ のペアが複数

```math
\{(x_{n}, d_{n})\}_{n=1, ..., N} ≡ \{ (x_{1}, d_{1}),(x_{2}, d_{2}), ..., (x_{N}, d_{N}) \}
```

のように与えられるとする。なお、これらのペア $(x_{n}, d_{n})$ 一つ一つを **訓練サンプル(training sample)** と呼び、その集合を **訓練データ(training data)** と呼ぶ。<br>
<br>
　目的とする関数をうまく表現できるように、ネットワークが表す関数を $w$ を調整して訓練データに適合させることを考える。具体的には、すべてのペア $(x_{n}, d_{n})(n = 1, 2, ..., N)$ について、入力 $x_{n}$ を与えた時のネットワークの出力 $y(x_{n}; w)$ が、なるべく $d_{n}$ に近づくように $w$ を調整する。これを **教師あり学習(supervised learning)** という。<br>
　このとき、 $x_{n}$ 　に対する出力 $y(x_{n}; w)$ と訓練データ $d_{n}$ との差を計る必要があり、それを行うのが **損失関数(loss function)** である。このようにして、訓練データに対して計算される損失を小さくするようにパラメータを定める考え方を **経験損失の最小化(empirical risk minimization)** と呼ぶ。<br>
　代表的な例のひとつとして、連続値を予測する回帰問題について考える。

### 5.回帰
　**回帰(regression)** あるいは回帰分析とは、二つの変数の関係をデータから推定することをいう。そのような関係を一つの関数 $y=f(x)$ とみて、これをニューラルネットワークで表現し、その関数の入出力のペアの事例 $\{(x_{n}, d_{n})\}_{n=1,...,N}$ を使って学習を行う。その結果、与えられた新しい入力 $x$ に対し、それが対応する出力 $y$ を予測できるようになる。<br>
　ネットワークの出力層は、目的とする関数の出力 $y$ を与えられるように設計する。そうして出力層の活性化関数を選んだうえで、 $x_{n}$ に対するネットワークの出力 $y(x_{n}; w)$ が、訓練データの目標出力 $d_{n}$ に可能な限り近くなるようにすることを考える。その「近さ」の尺度、すなわち損失関数には、例えば二乗誤差

```math
||d_{n} - y(x_{n};w)||^{2}
```

を使う。これを訓練データの $N$ 個のサンプルについて加算した

```math
E(w) = \frac{1}{2}\sum_{n=1}^N||d_{n} - y(x_{n}; w)||^{2}
```

が最も小さくなるように $w$ を決める。なお、二乗誤差は回帰に用いられる損失関数として一般的だが、いつもそうすべきというわけではない。

## 6. 確率的勾配降下法
　順伝播型ネットワーク学習は、訓練データ

```math
D = \{(x_{1}, d_{1}), (x_{2}, d_{2}), ..., (x_{N}, d_{N})\}
```

に対して計算される損失関数 $E(w)$ を、 $w$ について最小化することに帰着される。可能であれば大域的最小点 $w=argmin_{w}E(w)$ を求めたいが、複雑さから困難であり、 $E(w)$ の局所的な極小点 $w$ を得ることしかできない。しかし、この極小点での損失関数の値がある程度小さければ、目的とする推論をうまく実行できる可能性がある。<br>
　ニューラルネットワークの学習では、 **勾配降下法(gradient descent method)** を用いる。 **勾配(gradient)** は損失関数の一次微分であり

```math
\nabla{E} ≡ \frac{\partial E}{\partial w} = [\frac{\partial E}{\partial w_{1}} ・・・\frac{\partial E}{\partial w_{M}} ]^{T}
```

というベクトルである( $M$ は $w$ の成分後)。勾配降下法は他の最小化手法と同様に反復法であり、 $w$ を繰り返し更新することで極小解を探索する。一度の更新で、 $w$ を負の勾配方向( $-\nabla{E}$ )に少しだけ動かす。つまり現在の重みを $w_{t}$ 、動かした後の重みを $w_{t+1}$ とすると

```math
w_{t+1} = w_{t} - \varepsilon \nabla E
```

となる。ここで $\varepsilon$ は $W$ の更新量の大きさを定める定数で、 **学習率(learning rate)** と呼ぶ。<br>
　$\varepsilon$ が十分小さければ(かつ $E$ が十分滑らかなら)、 $t$ の増加に伴って $E(w_{t})$ は減少するので、更新を繰り返せばいつかは極小点になる。<br>

### SGD
　最小化した損失関数 $E(w)$ は、各サンプル一個に関する損失 $E(n)$ の和として

```math
E(w) = \sum_{n=1}^{N}E_{n}(w)
```

と与えられる。この $E(w)$ の勾配を用いて $w$ を更新する方法を、 **バッチ学習(batch learning)** と呼ぶ。<br>
　これに対し、サンプル一つの損失 $E_{n}$ を使い、パラメータを更新する方法がある。この方法は **確率的勾配降下法(stochastic gradient descent)** と呼ばれ、頭文字をとり **SGD** と略される。一つサンプル $n$ について計算される損失 $E_{n}(w)$ の勾配 $\nabla E_{n}$ を計算し、 $w$ を更新する。

```math
w_{t+1} = w_{t} - \varepsilon \nabla E_{n}
```

　次の更新の際は、別のサンプル $n'$ を取り出し、それの損失 $E_{n'}$ とその勾配 $\nabla E_{n'}$ を計算し、 $w_{t+1}$ を更新する。SGDでは、このように $w$ の更新のたびにサンプルを取り変える。

## 7. 誤差逆伝播法
　ニューラルネットワークの学習には、勾配降下法をベースとする方法を用いるが、損失関数 $E(w)$ の勾配 $\nabla E = \frac{\partial E(w)}{\partial w}$ の計算を必要とする。勾配のベクトルの各成分は、各層の結合重みと各ユニットのバイアスでの損失関数の微分( $\frac{\partial E}{\partial w_{ji}}$ と $\frac{\partial E}{\partial b_{j}}$ )である。しかし、これらの微分の計算は中間層、特に入力に近い深い層のパラメータほど、計算の手間は多くなる。<br>
　**誤差逆伝播法(back propagation)** は、この計算を効率よく実行する方法である。<br>
　以降、表記を簡素化するため、 $+1$ をいつも出力する特別な第ゼロ番ユニットを各層に導入し、バイアス $b_{j}$ をそのユニットと各ユニット $j$ との結合重み $w_{0j}^{(l)} = b_{j}^{l}$ と考えることとする。つまり、 $l$ 層のユニットへの入力は、 $l-1$ 層の第 $0$ ユニットの出力が常に $z_{0}^{(l-1)} = 1$ となることで、

```math
u_{j}^{(l)} = \sum_{i=1}^{n}w_{ji}^{(l)}z_{i}^{(l-1)} + b_{j} = \sum_{i=0}^{n}w_{ji}^{(l)}z_{i}^{(l-1)}
```

と簡潔に書き表せる。

### 二層ネットワークでの計算
　二層ネットワークでの場合を考える。回帰問題への適用を念頭に、出力層の活性化関数を恒等関数とする。ただし、中間層のユニットは任意の活性化関数 $f$ を持つとする。このネットワークでは、入力 $x = [x_{1} x_{2} x_{3} x_{4}]^{T}$ は出力へと向けて次のように伝播する。入力層の出力を $z_{i}^{(1)} = x_{i}$ とすると、中間層の出力は

```math
z_{j}^{(2)} = f(u_{j}^{(2)}) = f(\sum_{i}w_{ji}^{(2)}z_{i}^{(1)})
```

となる。出力層の活性化関数は恒等写像なので、その出力は以下の通りである。

```math
y_{j}(x) = z_{j}^{(3)} = u_{(j)} = \sum_{i}w_{ji}^{(3)}z_{i}^{(2)}
```

　このネットワークの損失関数に二乗誤差

```math
E_{n} = \frac{1}{2}||y(x) - d||^{2} = \frac{1}{2}\sum_{j}(y_{j}(x) - d_{j})^{2}
```

を選んだ時、重みでの微分( $\frac{\partial E_{n}}{\partial w_{ji}^{(3)}}$ と $\frac{\partial E_{n}}{\partial w_{ji}^{(2)}}$ )はどのように計算されるか。<br>
　まず、出力層の重みに関する微分 $\frac{\partial E_{n}}{\partial w_{ji}^{(3)}}$ は比較的簡単に計算ができる。これは

```math
\frac{\partial E_{n}}{\partial w_{ji}^{(3)}} = (y(x) - d)^{T}\frac{\partial y}{\partial w_{ji}^{(3)}}
```

のように展開でき、右辺の $\frac{\partial y}{\partial w_{ji}^{(3)}}$ はベクトルだが、$j$ 成分のみが $z_{i}^{(2)}$ で、それ以外の成分が $0$ なので

```math
\frac{\partial y}{\partial w_{ji}^{(3)}} = [0 ・・・ 0 z_{i}^{(2)} 0 ・・・ 0]^{T}
```

の形になる。従って

```math
\frac{\partial E_{n}}{\partial w_{ji}^{(3)}} = (y_{j}(x) - d_{j})z_{i}^{(2)}
```

のように求まる。<br>
　次に、中間層の重みに関する微分 $\frac{\partial E_{n}}{\partial w_{ji}^{(2)}}$ を計算する。 $w_{ji}^{(2)}$ は $u_{j}^{(2)} = \sum_{i}w_{ji}^{(2)}z_{i}^{(1)}$ のように $u_{j}^{(2)}$ のなかにのみ存在する。そこで、 $E_{n}$ を $u_{j}^{(2)}$ で微分し、連鎖則を用いると

```math
\frac{\partial E_{n}}{\partial w_{ji}^{(2)}} = \frac{\partial E_{n}}{\partial u_{j}^{(2)}}\frac{\partial u_{j}^{(2)}}{\partial w_{ji}^{(2)}}
```

となる。 $u_{j}^{(2)} = \sum_{i}w_{ji}^{(2)}z_{i}^{(1)}$ であることを使うと、上記の式の右辺第二項は直ちに

```math
\frac{\partial u_{j}^{(2)}}{\partial w_{ji}^{(2)}} = z_{i}^{(1)}
```

と計算される。<br>
　連鎖則を使用した式の右辺第一項 $\frac{\partial E_{n}}{\partial u_{j}^{(2)}}$ は、次のように計算する。中間層のユニット $j$ への入力 $u_{j}^{(2)}$ は、このユニットとつながりをもつ出力層のユニット $k(=1, 2, 3)$ に伝わるので、これら三つのユニットはそれぞれの総入力 $u_{k}^{(3)}$ に影響する。 $E_{n}$ は $u_{k}^{3}(k = 1,2,3)$ の関数であるため、 $E_{n}$ の $u_{j}^{(2)}$ に関する微分は、連鎖則を使うと、これらの出力層の三ユニットについてのすべての和

```math
\frac{\partial E_{n}}{\partial u_{j}^{(2)}} = \sum_{k}\frac{\partial E_{n}}{\partial u_{k}^{(3)}}\frac{\partial u_{k}^{(3)}}{\partial u_{j}^{(2)}}
```

で表現できる。右辺の $\frac{\partial E_{n}}{\partial u_{k}^{3}}$ は、 $E_{n} = \frac{1}{2}\sum_{k}(y_{k}(x) - d_{k})^{2} = \frac{1}{2}\sum_{k}(u_{k}^{(3)} - d_{k})^{2}$ から

```math
\frac{\partial E_{n}}{\partial u_{k}^{(3)}} = u_{k}^{(3)} - d_{k}
```

と計算でき、 $\frac{\partial u_{k}^{(3)}}{\partial u_{j}^{(2)}}$ は $u_{k}^{3} = \sum_{j} w_{kj}^{3}f(u_{j}^{2})$ であることから

```math
\frac{\partial u_{k}^{(3)}}{\partial u_{j}^{(2)}} = w_{kj}^{(3)}f'(u_{j}^{(2)})
```

と計算できる。これらを中間層の重みに関する微分 $\frac{\partial E_{n}}{\partial w_{ji}^{(2)}}$ の式に代入すると

```math
\frac{\partial E_{n}}{\partial w_{ji}^{(2)}} = (f'(u_{j}^{(2)})\sum_{k}w_{kj}^{(3)}(u_{k}^{(3)} - d_{k}))z_{i}^{(1)}
```

を得る。
